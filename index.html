Инструкция

1

Для Windows

Разархивируй АРХИВ по пути:

Мои документы/Sports Interactive/Football Manager 20**/graphics/
1.1

Для MacOS

Разархивируй АРХИВ по пути:

Библиотека/Application Support/Sports Interactive/Football Manager 20**/graphics/

ЕСЛИ папки graphics нет, то СОЗДАЙ её

2

Настройки в игре

Зайди в FM и в настройках игры на вкладке:

  • "Общие" - "Расширенный" - "Интерфейс"
  • Тема: "Использовать кэширование для снижения времени загрузки страниц" - снять галочку
  • "Обновить скин после подтверждения изменений в настройках" - поставить галочку
